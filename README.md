# PAPERIO

## Get started

To start local runner you have run following commands

- `virtualenv env`
- `source ./env/bin/activate`
- `pip install pipenv`
- `pipenv install`
- `python localrunner.py`

IF you would like to run your bot implementation here is command:

```bash
 python localrunner.py -p1 "python bot.py"

```
